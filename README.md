# MediNearby

MediNearby is a GitHub-ready medicine reminder website built with **HTML, CSS, and JavaScript**.
It lets users add medicine name, dosage, food timing, reminder times, and notes. The app stores all data in the browser using **localStorage**, so no backend is required.

## Features

- Add medicine name
- Add dosage
- Add how to eat / food timing
- Add multiple reminder times
- Start date and end date
- Browser notification reminders
- Mark medicine as taken
- Delete medicine reminders
- Search medicines
- Attractive Canva-style glassmorphism UI
- Works as a static website on GitHub Pages

## Files

- `index.html` → website structure
- `style.css` → design and responsive layout
- `script.js` → medicine logic, local storage, reminders
- `README.md` → project details

## How to use locally

1. Download or extract the project files.
2. Open `index.html` in your browser.
3. Click **Enable Notifications**.
4. Add medicine details and save.

## How to upload to GitHub

1. Create a new GitHub repository.
2. Upload these files:
   - `index.html`
   - `style.css`
   - `script.js`
   - `README.md`
3. Go to **Settings > Pages** in your repository.
4. Select the main branch and root folder.
5. Save the settings.
6. GitHub will generate your live website link.

## Important note

Notifications work when:
- the browser allows notifications
- the website tab is open in the browser

This version is frontend-only and does not need Node.js, Express, or a database.
